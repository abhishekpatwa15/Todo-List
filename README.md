# Todo List Application

A simple and intuitive Todo List application that helps users manage their tasks efficiently. Built with technologies like React, Node.js, MongoDB. This project allows users to create, read, update, and delete their tasks seamlessly.

## Features

- **Create Tasks**: Easily add new tasks to your list.
- **View Tasks**: See all your tasks at a glance with their statuses.
- **Update Tasks**: Edit existing tasks to reflect changes.
- **Delete Tasks**: Remove tasks that are no longer needed.
- **Mark as Complete**: Keep track of completed tasks.

## Technologies Used

- Frontend: React.js , CSS
- Backend: Node.js 
- Database: MongoDB
